Face Antispoofing (In progress)

Developing a technique that can distinguish the spoof face accesses from the genuine ones. This will reduce the possibility of a facial biometric system from being deceived or spoofed by non-real faces.  Snapchat's faceswap feature is used as the spoofing technique

Dataset: The dataset comprises of spoof videos of 31 subjects with photos of 20 subjects superimposed using Snapchat's faceswap feature (one frontal and one oblique). 

Techniques:

Using color and textural analysis as the liveliness indicators, extracted RGB color histograms from the image, using 8 bits per channel, normalized it to get the color features. Will be using local binary patterns LBP for textural analysis 

Product Recommender using Transfer Learning

Recognize clothing style and colour in apparel images using Deep Learning features and recommend visually similar products.

Dataset:  Amazon product data for including 142.8 million reviews for 9.4 million products, trimmed such that each user has at least 5 reviews. For image tagging: 400 images for each Product Type*Colour 

Functionality: 

  • Image tagging (Tagging an image to the class of products it belongs to) 
  • The user can click a photograph of the product that they like, the recommender would suggest similar products available in the store. 
  • While viewing a product, the user will be able to see similar products. Recommended products for the user based on viewing history and ratings
Techniques:
  • Values of the penultimate layer of the AlexNet neural network were calculated for each product and used as the deep features of the product
  • Used the ball tree method for nearest neighbour search based on Euclidean distance calculated using deep features of images to get visually similar products
  • Performed k-means clustering to group similar users based on their reviews
  • Created a transfer learning-based classifier for image tagging
